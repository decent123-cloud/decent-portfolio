# Emmanuel Decent's Portfolio

A modern, responsive personal portfolio website showcasing projects, skills, and professional information.

## 🌟 Features

- **Responsive Design**: Fully responsive on mobile, tablet, and desktop devices
- **Modern UI**: Clean, professional design with smooth animations
- **Hero Section**: Engaging landing section with call-to-action buttons
- **About Section**: Personal bio and statistics
- **Skills Section**: Organized by categories (Frontend, Backend, Databases, Tools)
- **Projects Section**: Featured projects with descriptions and tech stacks
- **Contact Form**: Easy way for visitors to get in touch
- **Smooth Scrolling**: Smooth navigation between sections
- **Mobile Menu**: Hamburger menu for mobile devices

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.)
- Git (optional, for version control)

### Installation

1. **Clone the repository** (or download the files):
```bash
git clone https://github.com/emmanueldecent/decent-potfolio.git
cd decent-potfolio
```

2. **Open the portfolio**:
   - Simply open `index.html` in your web browser
   - Or use a live server extension in VS Code:
     - Install "Live Server" extension
     - Right-click on `index.html` and select "Open with Live Server"

## 🎨 Customization

### Update Your Information

1. **Open `index.html`** and update:
   - Your name in the hero section
   - Bio and description
   - Contact email
   - Social media links
   - Projects (replace sample projects with your own)
   - Skills based on your expertise

2. **Update Colors** (Optional):
   - Open `styles.css`
   - Modify CSS variables in `:root` section:
   ```css
   :root {
       --primary-color: #6366f1;    /* Change to your preferred color */
       --secondary-color: #8b5cf6;
       /* ... other colors ... */
   }
   ```

3. **Add Your Projects**:
   - In the Projects section, replace sample projects with your own
   - Update GitHub links to your actual project repositories
   - Modify tech stacks and descriptions

4. **Update Contact Email**:
   - Find `emmanuelabershi483@gmail.com` in the file
   - Replace with your actual email address

## 🚢 Deployment

### Deploy to GitHub Pages (Free & Easy)

1. **Push to GitHub**:
```bash
git add .
git commit -m "Initial portfolio commit"
git push origin main
```

2. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll to "GitHub Pages" section
   - Select `main` branch as source
   - Save and your site will be live at: `https://emmanueldecent.github.io/decent-potfolio`

### Deploy to Vercel (Free & Fast)

1. **Sign up at Vercel**: https://vercel.com
2. **Connect your GitHub repository**
3. **Deploy** - Vercel will automatically deploy on every push
4. **Custom Domain** (optional) - Add your domain in Vercel settings

### Deploy to Netlify (Free & Easy)

1. **Sign up at Netlify**: https://netlify.com
2. **Drag and drop your folder** or connect GitHub
3. **Deploy** - Your site will be live immediately
4. **Custom Domain** (optional) - Configure in Netlify settings
